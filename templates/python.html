Python

Basics

Let's start from the beginning: how do you create a variable? To create a variable in Python, write: 

name = "Alex"
age = 16

Variable names cannot use Python keywords such as if, for, while, class, etc.

Conditions

Python uses if, elif and else to make decisions. 

age = 18

if age >= 18:
    print("Adult")
else:
    print("Minor")

Input Function

name = input("What's your name?")
print("Hello,", name)

input() always returns text (string).

Useful Functions

len("Hello") #string length is 5 characters
abs(-5) #will output the absolute value of the number -5 — 5
type(10) #will output < class 'int' >

Calculator Example

a = int(input("First number: "))
b = int(input("Second number: "))
op = input("Operation: ")

if op == "+":
    print(a + b)
elif op == "-":
    print(a - b)
elif op == "*":
    print(a * b)
elif op == "/":
    print(a / b)
else:
    print("Error")
